Your Data Stays on Your Desk

OpenClaw runs locally on your hardware. Your emails, calendar, files, and credentials never leave your machine. We follow the official hardening guide to lock everything down:

  • Locked network access: Loopback-only binding, no public endpoints, DM pairing for messaging
  • Restricted tool permissions: Sandboxing enabled, elevated tools gated, explicit allowlists for sensitive actions
  • Encrypted credential storage: All API keys and passwords locked to your user account with proper file permissions
  • Security audit before handoff: We run the full security checklist. You can re-run it anytime.
